# Location Spec: Sector 3 (The Deep Station)
**Theme**: The Recursive Core (The Loop)
**Design Law**: "The End is the Beginning"

---

## 1. Sensory Pillars
*   **Visual**:
    *   *Lighting*: Stark, clinical white mixed with emergency red strobes.
    *   *Palette*: Sterile White, Blood Red, Void Black.
    *   *Architecture*: Infinite server racks, glass walkways over abysses, mirrors.
*   **Audio**:
    *   *Ambience*: Absolute silence punctuated by deafening alarm blasts. Heartbeat.
    *   *Music*: Shepard Tone (Always rising tension), distorted choir.
*   **Smell/Touch**:
    *   *Smell*: Sterile antiseptic, burning dust.
    *   *Touch*: Cold glass, vibrating floors.
