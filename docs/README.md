## Key Principles

From `00_PHILOSOPHY_FOUNDATION.md`:

1. **Feel Comes First** - 30-second test
2. **Respect Player Intelligence** - No tutorials
3. **Every Element Serves Multiple Purposes**
4. **Dialogue-Driven Immersion** - Everything in narrative container
5. **UI Consolidation** - 7 elements max
6. **Joyce's Scrupulous Meanness** - Every word earns its place
7. **Demonstrate Before Explain** - Show orb, then Samuel explains
8. **Clean Before Add** - Remove bloat before adding features
